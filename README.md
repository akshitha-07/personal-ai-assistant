# Neia

> Your daily AI assistant.

Neia is an AI-powered productivity assistant designed to help users organize tasks, manage daily life, remember important information, and stay productive through a simple and intelligent interface.

## Features

* AI chat
* Productivity assistance
* Task management
* Smart reminders
* Memory system
* Personalized interactions

## Tech Stack

* React
* Vite
* Node.js
* Express
* Gemini API
* MongoDB

Currently in active development.
